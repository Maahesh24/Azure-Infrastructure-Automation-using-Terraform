# Azure Infrastructure using Terraform 🚀

## Project Overview
This project demonstrates Infrastructure as Code (IaC) using Terraform on Microsoft Azure.

## Tools Used
- Terraform
- Microsoft Azure
- GitHub

## What This Project Does
- Creates Resource Group
- Creates Virtual Network & Subnet
- Deploys Linux Virtual Machine

## How to Run

terraform init
terraform plan
terraform apply

## Outcome
- Automated infrastructure provisioning
- Scalable and repeatable deployment
